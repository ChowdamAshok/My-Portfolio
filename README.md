## 🚀 Overview

This portfolio reflects my journey as a **Java Full Stack Developer** with hands-on **AI/LLM integration** experience — a key differentiator at the fresher level. It highlights projects built with Spring Boot, React.js, and PostgreSQL, several of which integrate Groq's LLaMA 3.3 models for chatbots, smart search, and review summarization. The site itself is fully responsive, GSAP-animated, and supports light/dark themes.

### 🔑 Key Features

- Clean, modern UI with smooth GSAP scroll animations
- Dark / light mode toggle with saved preference
- Mobile-first responsive layout with animated hamburger navigation
- Smooth in-page scrolling via GSAP `ScrollToPlugin`
- Skill tags organized by category (Backend, Frontend, Databases, AI & GenAI, Salesforce, DevOps)
- Featured vs. early-work project sections (collapsible)
- Interactive timeline for experience and education
- Working contact form (Formspree integration)
- Back-to-top button with scroll-aware visibility
- Certifications showcase with badge gallery

## 📁 Usage

To run the project locally:

```bash
git clone https://github.com/ChowdamAshok/My-Portfolio.git
cd My-Portfolio
```

Then simply open `index.html` in your browser, or serve it with any static file server (e.g. the VS Code "Live Server" extension) for the best experience with relative asset paths.
